# SkillCypher

**SkillCypher** is a full-stack learning platform for delivering structured programming courses and managing student batches through a modern web interface.

**Live Demo:** https://skillcypher.tech

---

## Tech Stack

**Frontend**
- React
- Vite
- Tailwind CSS
- React Router

**Backend**
- Node.js
- Express.js
- MongoDB (Mongoose)

**Tools & Services**
- JWT Authentication
- Nodemailer / Resend
- Multer
- Heroku Deployment

---

## Features

- Course & batch management
- Student learning dashboard
- REST API based architecture
- Secure authentication
- Responsive UI

---

## Run Locally

### Backend
```bash
cd Backend
npm install
npm run dev
````

### Frontend

```bash
cd IMAGINO_MAIN
npm install
npm run dev
```

Frontend → [http://localhost:5173](http://localhost:5173)
Backend → [http://localhost:8080](http://localhost:8080)
