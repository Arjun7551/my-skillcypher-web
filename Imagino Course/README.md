# Quick Start: Imagino Course

## Install and Run

1. **Install backend dependencies:**
   ```powershell
   cd Backend
   npm install
   ```

2. **Install frontend dependencies:**

   ```powershell
   npm install
   ```

3. **Start backend server:**
   ```powershell
   cd ../Backend
   npm start
   ```

4. **Start frontend dev server:**
   ```powershell
   cd "../IMAGINO MAIN"
   npm run dev
   ```

- Make sure MongoDB is running and environment variables are set in `Backend/.env` if needed.
